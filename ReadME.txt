Q. AI를 사용하여 만들어진 바이브 코딩들 아닌가?

일정 부분은 맞습니다. 가령 전투에 사용된 코드 중 Assets/Scenes/InSeong/Scripts/Units/Battle/BattleBase.cs 파일 같은 경우엔 상당 부분 AI를 통한 코드를 사용했습니다.
예를 들면 - 주석에  // ★ 이미 고정된 타겟이 있고 유효하면 바로 Moving으로 ★ 와 같이 ★ 기호가 있는 부분은 AI를 통한 코드 초안을 사용했거나, 작성된 코드가 AI를 통해 수정된 흔적입니다.
또한, 제가 매커니즘 및 기초 알고리즘을 작성한 토대로 만들어진 초안을 사용하기도 했습니다.

그러나 대부분의 코드는 제가 직접 작성 또는 완성했으며, AI 코드는 대체로 이렇게 초안이 작성되었습니다.

예)
//무언가를 실행하는 메서드
void Do_Smth(int args1){
  //1. 인자 args1을 받아서 ...
  //2. 그 다음 실행할 내용...
  //3. 그 다음 결과를 반환
}
와 동일한 형식으로 메서드를 작성할 때 '기능을 완성하기 위해서 필요한 사안' 실제 코드 없이 구어체 주석으로만 초안을 작성하여 사용하여 최대한 구현했고,
버그를 찾거나 고치지 못한 부분들로 사용을 최소화 하려 했습니다.
